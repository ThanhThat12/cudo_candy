# Hướng dẫn thêm hình ảnh cho Blog

Để hiển thị hình ảnh cho các bài viết blog, vui lòng thêm các file ảnh sau vào thư mục `public/images/blog/`:

## Danh sách hình ảnh cần thêm:

1. **cudon-origin.jpg** - Hình ảnh về nguồn gốc Cu đơ Hà Tĩnh (truyền thống, làng nghề)
2. **real-sugarcane.jpg** - Hình ảnh mật mía thật, mía tươi
3. **gift-cudon.jpg** - Hình ảnh Cu đơ đóng gói làm quà tặng
4. **storage.jpg** - Hình ảnh về bảo quản Cu đơ
5. **health-benefits.jpg** - Hình ảnh về sức khỏe, dinh dưỡng
6. **recipes.jpg** - Hình ảnh các món ăn từ Cu đơ

## Kích thước đề xuất:
- Tỷ lệ: 16:9 hoặc 4:3
- Độ phân giải: tối thiểu 1200x800 pixels
- Định dạng: JPG hoặc PNG

## Lưu ý:
- Tất cả hình ảnh nên có chất lượng cao
- Hình ảnh nên liên quan trực tiếp đến nội dung bài viết
- Có thể sử dụng hình ảnh minh họa tạm thời từ các nguồn miễn phí như Unsplash, Pexels
